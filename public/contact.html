FAQ

Is the Free plan really free forever?
Yes. The free tier has no expiration date. 500 requests/day is enough for most personal projects and bots.
Can I self-host CrazyApi?
Absolutely. The full source code is available. Clone it, configure Supabase and deploy anywhere. See the self-hosting docs.
What happens if I hit the rate limit?
You'll receive a 429 Too Many Requests response. Wait a minute and try again, or upgrade to Pro for higher limits.
Do you store my API requests?
We store only the request count per key for rate limiting. We don't log query content or results.
